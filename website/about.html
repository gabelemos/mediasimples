O "MédiaSimples"

É um projeto que tem como intuito facilitar a visualização das
notas bimestrais/médias dos estudantes.

Tecnologias Utilizadas

Python

Utilizei Python, pois é uma linguagem que me permitiu articular bem como seria o projeto, e pensar maneiras diferentes para executar a ideia que eu tinha em mente, que era como dito acima, um software que procura ajudar a tornar bem mais prático a visualização e os cálculos das médias bimestrais dos estudantes que utilizam a plataforma chamada "SIEPE" para visualizar seus boletins.

Selenium

Para fazer WebScraping, utilizei em sua maioria, uma biblioteca também disponível para Python chamada "Selenium". Ela tem como foco principalmente ajudar a obtenção de dados da WEB por meio de scripts com programação, ajudando desenvolvedores no processo de automatização de alguns processos, assim como esse de cálculos das notas.

WebScraping: Técnica para adquirir informações expostas na WEB.

Tkinter

Após realizar a obtenção dos dados via Selenium, utilizei outra biblioteca famosa para Python chamada TkInter para assim conseguir visualizar melhor os dados, e também tornar mais compreensível para o usuário, tanto os dados quanto suas opções dentro do software.
